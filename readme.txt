Descripción:
============

Spelling Warrior es un juego lúdico para el aprendizaje del inglés, inspirado por juegos como los antiguos Final Fantasy de 16 bits. Apela al factor nostalgia del jugador junto con elementos clásicos de los RPG.

La formulación de las preguntas se configura mediante un archivo JSON (questions.json).


Requisitos:
===========

Escritorio: 
- Internet Explorer 10+, Google Chrome 23+, Safari 7+ o Mozilla Firefox 20+

Móvil:
- Dispositivo Android 4+ con procesador Dual Core de 1.4Ghz o superior.
- Navegador Google Chrome.


Instrucciones de Uso / Instalación
===================================

Uso Local:
 1. Crear un servidor simple http desde la carpeta "dist". (NodeJS, Python, etc).
 2. Arrancar aplicación desde el navegador
